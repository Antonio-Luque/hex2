6DD7: Get Hexagon Screen Address
Used by the routines at Draw_Board and Draw_Token.
Get the screen address of an hexagon on board.
Input
BC Hexagon board indexes (B=column, C=row)
Output
HL Hexagon screen address
Draw_GetHEX_Screen 6DD7 LD DE,$6198 DE points to BOARD_MAP-8 address (adjusted for index 0)
6DDA CALL Main_Search_Table search the screen address of the hexagon in table
6DDD LD L,(HL) set the low byte of the screen address in L
6DDE LD A,(DE) set the high byte of the screen address in H
6DDF LD H,A
6DE0 RET
