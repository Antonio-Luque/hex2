7918: Apply Color to Frame
Used by the routines at Screen_Menu and Main_New_Level.
Set up frame color and print level number depending on current level.
Input
HL GAME_LEVEL
Screen_Frame_Color 7918 LD B,$00 set game level in BC
791A LD C,(HL)
791B LD A,$30 convert game level number to game level character
791D ADD A,C
791E LD HL,$400F HL points to screen address of game level
7921 CALL Print_Char print game level
7924 LD HL,$620F HL points to ROW_COLOR_MAP-1 (adjusted for index 0)
7927 ADD HL,BC use BC as index for color map address
7928 LD A,(HL) A=level color
7929 CALL Screen_Tiles_Color color the frame according to the current level
792C AND $07 mask color
792E LD ($619C),A save at LEVEL_COLOR
7931 RLA rotate to text level color
7932 RLA
7933 RLA
7934 LD HL,$5808 HL points to attribute address of level text
7937 LD B,$09 length of level text (loop counter)
Screen_Level_Loop 7939 LD (HL),A apply color to level text
793A INC L move HL to attribute address of next character of level text
793B DJNZ Screen_Level_Loop repeat for each character of level text
793D RET
