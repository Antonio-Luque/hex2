6F65: Print Char
Used by the routines at Print_String, Screen_Frame_Color, Screen_How2Play_CYAN, Screen_How2Play_LevelUp and Input_Move.
Print a character on the screen.
Input
A Character to print (ASCII)
HL Character screen address
Print_Char 6F65 LD DE,$6800 DE points to FONT-256 address
This entry point is used by the routine at Screen_Frame with A=tile index and DE pointing to FRAME_TILES.
Print_Tile 6F68 PUSH HL store pointer to character/tile screen address
6F69 LD H,$00 set character/tile index in HL
6F6B LD L,A
6F6C ADD HL,HL a character/tile is 8 bytes long
6F6D ADD HL,HL
6F6E ADD HL,HL
6F6F ADD HL,DE move HL to character/tile address
6F70 EX DE,HL DE points to character/tile address
6F71 POP HL restore pointer to character/tile screen address
6F72 PUSH BC store BC register
6F73 LD C,H save high byte of character/tile screen address
6F74 LD B,$08 number of character/tile bytes (loop counter)
Print_Tile_Pixels 6F76 LD A,(DE) get character/tile byte
6F77 LD (HL),A put character/tile byte on screen
6F78 INC DE move DE to next character/tile byte
6F79 INC H move HL down one pixel-line
6F7A DJNZ Print_Tile_Pixels repeat for each character/tile bytes
6F7C LD H,C restore high byte of character/tile screen address
6F7D POP BC restore BC register
6F7E RET
