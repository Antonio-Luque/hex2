7781: Draw Screen Frame
Used by the routines at Screen_Menu and Main_Start.
Draw the screen frame on the screen.
Screen_Frame 7781 LD DE,$6229 DE points to STR_LEVELTEXT address ("Level")
7784 LD HL,$4009 HL points to screen address of the string
7787 CALL Print_String print "Level"
778A CALL Screen_Vertical_Bars draw the vertical bars of frame
778D LD BC,$628E BC points to FRAME_MAP address
7790 LD HL,$4000 HL points to the screen address of the up-border
7793 CALL Screen_Frame_Border draw up-border
7796 LD HL,$50E0 HL points to the screen address of the bottom-border
Screen_Frame_Border 7799 LD IXl,$20 border width in bytes (loop counter)
Screen_Frame_Loop 779C LD A,(BC) get tile index
779D CP $FF is it -1?
779F JR Z,Screen_Frame_Next yes, there is no tile to print
77A1 LD DE,$650E DE points to FRAME_TILES address
77A4 CALL Print_Tile print frame tile
Screen_Frame_Next 77A7 INC L HL points to the next screen address (next column)
77A8 INC BC BC points to the next tile index
77A9 DEC IXl decrement loop counter
77AB JR NZ,Screen_Frame_Loop repeat for each tile
77AD LD IX,$5807 IX points to the "concave left" tile attribute address
77B1 LD (IX+$00),$08 color tile (BLUE PAPER)
77B5 LD (IX+$0A),$08 color "concave right" tile
77B9 LD IX,$5AF1 IX points to the "convex left" tile attribute address
77BD LD (IX+$00),$01 color tile (BLUE INK)
77C1 LD (IX+$04),$01 color "convex right" tile
77C5 RET
