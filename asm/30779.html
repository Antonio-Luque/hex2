783B: Apply Color to Statics Items
Used by the routine at Main_Start.
Apply color to some statics items of the screen.
Screen_Color_Statics 783B LD IX,$5878 IX points to the attr. address of the cyan player timer
783F LD C,$07 set color in C (WHITE)
7841 CALL Screen_Apply_Color apply color
7844 LD IXl,$98 IX points to the attr. address of the cyan player name
7847 LD C,$05 set color in C (CYAN)
7849 CALL Screen_Apply_Color apply color
784C LD IX,$5918 IX points to the att. address of the cyan player messages
7850 LD C,$47 set color in C (BRIGHT+WHITE)
7852 CALL Screen_Apply_Color apply color
7855 LD IXl,$E2 IX points to the attr. address of the red player timer
7858 LD C,$07 set color in C (WHITE)
785A CALL Screen_Apply_Color apply color
785D LD IX,$5A02 IX points to the attr. address of the red player name
7861 LD C,$02 set color in C (RED)
7863 CALL Screen_Apply_Color apply color
7866 LD IXl,$82 IX points to the attr. address of the red player messages
7869 LD C,$47 set color in C (BRIGHT+WHITE)
This entry point is used by the routines at Print_Clear_Alert, Print_Alert, Print_Wait, Print_Clear_Move, Print_Move, Screen_Menu, Screen_How2Play_RED and Screen_How2Play_CYAN with C=color attribute and IX pointing to string attribute address.
Screen_Apply_Color 786B LD B,$06 number of bytes to be colored (loop counter)
This entry point is used by the routines at Screen_Menu, Screen_How2Play_RED, Screen_How2Play_CYAN, Screen_How2Play_LevelUp, Screen_Menu_Credits and Screen_Last_Level with B=number of bytes, C=color attribute and IX pointing to string attribute address.
Screen_Apply_Color_Loop 786D LD (IX+$00),C apply color
7870 INC IXl move IX to the next attribute address
7872 DJNZ Screen_Apply_Color_Loop repeat for each byte
7874 RET
