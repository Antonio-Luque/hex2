6D74: Search Element in Table
Used by the routines at Draw_GetHEX_Screen and Main_GetPlayer_Move.
Search an element in a 8x8 bytes table.
Input
BC Element indexes (B=column, C=row)
DE Table address in which to search for the element
Output
DE Address of first element in the C row of the table
HL Address of the element in the BC indexes of the table
First, get the C row in table.
Main_Search_Table 6D74 LD H,$00 set row index in HL
6D76 LD L,C
6D77 ADD HL,HL a row is 8 bytes long
6D78 ADD HL,HL
6D79 ADD HL,HL
6D7A ADD HL,DE HL points to the address of C row
6D7B EX DE,HL save it in DE
Now, go for to the B column.
6D7C LD H,$00 set column index in HL
6D7E LD L,B
6D7F ADD HL,DE HL points to the address of BC indexes
6D80 RET
