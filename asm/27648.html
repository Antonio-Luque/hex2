6C00: Main Entry Point
Main entry point of the program.
Main_Entry 6C00 DI disable interrupts
6C01 LD HL,$761D HL points to Timer_Interrupt address
6C04 LD (IY+$67),$C3 set 'JP' opcode at JUMP_INT
6C08 LD (IY+$68),L set low byte of Timer_Interrupt address at JUMP_INT+1
6C0B LD (IY+$69),H set high byte of Timer_Interrupt address at JUMP_INT+2
6C0E LD A,$2E set high byte of interrupt vector address
6C10 LD I,A set interrupt vector address
6C12 IM 2 set interrupt mode 2
6C14 LD SP,$6000 move stack
6C17 XOR A A=0
6C18 OUT ($FE),A set border color (BLACK)
6C1A CALL Screen_Menu show main menu and wait for an option
This routine continues into Main_Start.
