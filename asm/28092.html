6DBC: Animate Hexagon Sprite
Used by the routines at Main_Match_Loop and PathFinder_BrightPath.
Animate an hexagon sprite on the screen.
Input
BC Hexagon board indexes (B=column, C=row)
Draw_Token 6DBC CALL Draw_GetHEX_Screen get hexagon screen address
6DBF LD BC,$0604 B=number of sprites, C=sprite number to change color
6DC2 LD DE,$659E DE points to HEXAGON_SPRITE address
Draw_Token_Loop 6DC5 PUSH HL store hexagon screen address
6DC6 CALL Draw_Sprite_Hexagon draw hexagon sprite
6DC9 POP HL restore hexagon screen address
6DCA LD A,B check loop counter
6DCB CP C is it 4?
6DCC CALL Z,Draw_Hexagon_Color yes, apply color to hexagon area
The value of A is initialized by the routine at Main_Start_Match and updated by the routines at Main_Match_End.
Draw_Token_Delay 6DCF LD A,$00 set delay duration
6DD1 CALL Main_Delay execute delay
6DD4 DJNZ Draw_Token_Loop repeat for each hexagon sprite
6DD6 RET
