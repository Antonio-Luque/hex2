77D9: Draw Fixed Tokens
Used by the routine at Main_Start.
Draw the fixed tokens on the screen.
Screen_Tokens 77D9 LD A,$05 set token color (CYAN)
77DB LD HL,$44BA HL points to the fixed cyan token screen address
77DE CALL Screen_Tokens_Draw draw cyan fixed token
77E1 LD A,$02 set token color (RED)
77E3 LD HL,$5424 HL points to the fixed red token screen address
Screen_Tokens_Draw 77E6 LD ($619E),A save token color at TOKEN_COLOR
77E9 CALL Draw_Hexagon_Color color fixed token area
77EC LD (IX+$40),A the fixed tokens are 3 bytes (attributes) high, so we have to color the remaining 2 bytes that have not been colored
77EF LD (IX+$41),A
77F2 LD DE,$663E DE points to HEXAGON_TOKEN address
77F5 JP Draw_Sprite_Hexagon draw fixed token
