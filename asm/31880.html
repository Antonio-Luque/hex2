7C88: Clean Text Area
Used by the routines at Screen_How2Play_RED, Screen_How2Play_CYAN and Screen_How2Play_LevelUp.
Clean the text attributes area of "How to play" screens.
Input
HL Text area attributes address
Screen_Clear_Text 7C88 LD BC,$0030 length of attributes area: width x height (in bytes)
Screen_ClearText_Loop 7C8B LD DE,$0018 displacement for each text row
7C8E ADD HL,DE add displacement to text area attributes address
7C8F LD D,H DE points to text area attributes address
7C90 LD E,L
7C91 LD (HL),$00 set attributes color (black) for 1st byte of current row
7C93 INC DE move DE to next byte attribute address
7C94 LDI set attributes color for the rest of bytes of current row
7C96 LDI
7C98 LDI
7C9A LDI
7C9C LDI
7C9E LDI
7CA0 LDI
7CA2 LDI
7CA4 JP PE,Screen_ClearText_Loop repeat for each row
7CA7 RET
