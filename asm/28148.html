6DF4: Draw Board Letter/Number
Used by the routines at Screen_Letters and Screen_Numbers.
Draw a letter/number sprite on the screen.
Input
DE Letter/number sprite address
HL Letter/number screen address
Output
DE Next letter/number sprite address
Draw_Sprite_LetterNum 6DF4 LD C,$0A letter/number sprite height in bytes (loop counter)
Draw_Sprite_Loop 6DF6 LD A,(DE) get sprite byte
6DF7 LD (HL),A put sprite byte on screen
6DF8 INC DE move DE to next sprite byte address
6DF9 CALL Draw_Down_PixelLine move HL down one pixel-line
6DFC DEC C decrement sprite height
6DFD JR NZ,Draw_Sprite_Loop repeat for each byte in sprite
6DFF RET
