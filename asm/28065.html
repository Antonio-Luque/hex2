6DA1: Draw Empty Board
Used by the routines at Screen_How2Play_RED, Screen_How2Play_CYAN and Main_Start_Match.
Draw an empty 7x7 hexagons board on the screen
Draw_Board 6DA1 LD A,$07 set empty hexagon color (WHITE)
6DA3 LD ($619E),A store at TOKEN_COLOR
6DA6 LD C,$07 7 rows
Draw_Board_Row 6DA8 LD B,$07 7 columns
Draw_Board_Column 6DAA CALL Draw_GetHEX_Screen get hexagon screen address
6DAD CALL Draw_Hexagon_Color apply color to hexagon area
6DB0 LD DE,$657E DE points to HEXAGON_EMPTY address
6DB3 CALL Draw_Sprite_Hexagon draw an empty hexagon at screen address
6DB6 DJNZ Draw_Board_Column repeat for each column
6DB8 DEC C decrement row
6DB9 JR NZ,Draw_Board_Row repeat for each row
6DBB RET
