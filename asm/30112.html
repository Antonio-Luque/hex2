75A0: Calculate Victory Path
Used by the routine at Main_Match_Loop.
Calculate current player victory path.
Output
A $FF, if there isn't a victory path
A Start position (-1) of the victory path, otherwise
HL CYAN_QUEUE_AREA or RED_QUEUE_AREA
PathFinder_VictoryPath 75A0 LD A,($619E) get TOKEN_COLOR
75A3 CP $05 is it cyan?
75A5 JR Z,PathFinder_VPath_Cyan yes, check victory path for cyan player
75A7 CALL PathFinder_Red_LeftRight execute Path_Finder RED from LEFT to RIGHT
75AA LD HL,$6445 HL points to RED_BOARD_LEFT_RIGHT+69 address
75AD CALL PathFinder_PrepareBoard prepare board to calculate victory path
75B0 JP PathFinder_Red_CheckPath calculate RED victory path and return
