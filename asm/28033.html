6D81: Convert Board Position to Indexes
Used by the routines at Main_GetSpeccy_Move and PathFinder_BrightPath.
Convert a board position into a board indexes.
Input
L Board position
Output
BC Board indexes (B=column, C=row)
Main_Board2Indexes 6D81 LD A,L set board position in A to compare later
6D82 LD HL,$628E HL points to last BOARD_COORDINATES address (+1)
6D85 LD C,$07 7 rows
Main_Board2Indexes_Row 6D87 LD B,$07 7 columns
Main_Board2Indexes_Col 6D89 DEC HL move HL to next board position address
6D8A CP (HL) is A the position?
6D8B RET Z yes, return board indexes
6D8C DJNZ Main_Board2Indexes_Col repeat for each column
6D8E DEC HL skip borders
6D8F DEC C decrement row
6D90 JR Main_Board2Indexes_Row loop until find position
