75BF: Setup Board for Victory Path
Used by the routine at PathFinder_VictoryPath.
Update input board, changing $43 (victory distance + friendly flag) to $01, everything else to $7E (ENEMY).
After executing PathFinder_Cyan_UpDown or PathFinder_Red_LeftRight, execute this routine and then execute PathFinder_Cyan_CheckPath or PathFinder_Red_CheckPath to find a victory path.
Coded by Einar Saukas.
Input
HL CYAN_BOARD_UP_DOWN+69 or RED_BOARD_LEFT_RIGHT+69
PathFinder_PrepareBoard 75BF LD A,$43 set victory distance + friendly flag
75C1 LD C,$07 7 rows
PrepareBoard_Row 75C3 LD B,$07 7 columns
PrepareBoard_Column 75C5 CP (HL) is it win condition?
75C6 LD (HL),$01 save $01 by default
75C8 JR Z,PrepareBoard_Next yes, next position
75CA LD (HL),$7E no, save $7E (ENEMY)
PrepareBoard_Next 75CC DEC L HL points to next board position
75CD DJNZ PrepareBoard_Column repeat for each column
75CF DEC L skip borders
75D0 DEC L
75D1 DEC C decrement rows
75D2 JR NZ,PrepareBoard_Row repeat for each row
75D4 RET
